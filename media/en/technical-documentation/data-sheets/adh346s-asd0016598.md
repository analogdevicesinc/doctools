The ADH346S is an absorptive Voltage Variable Attenuator (VVA) operating from DC - 8 GHz. It features an on-chip reference attenuator for use with an external op-amp to provide simple single voltage attenuation control, 0 to -3 V. The device is ideal in designs where an analog DC control signal must control RF signal levels over a 30 dB amplitude range.
